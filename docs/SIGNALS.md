# SIGNALS — 시그널 엔진 (Phase 3 활성)

## Phase 1 현황
인터페이스만. NotImplementedError.

## 가중치 (확정)
- 매수 강도 기준점: **+8**
- 매도 강도 기준점: **-3** (보수적, 손절 빠르게)

## 가중치 후보 (예시)
- RSI(14) < 30: +2
- RSI(14) > 70: -2
- MACD 골든크로스: +3
- MACD 데드크로스: -3
- 20일선 위: +1
- 60일선 위: +1
- 200일선 위: +1
- 5거래일 +5% 이상: -1 (단기 과열)
- 거래량 평균 2배+ 상승: +2
- 1시간 봉 동일 신호: +1

## 결합 규칙
- 합산 점수 ≥ +8 → 매수 후보
- 합산 점수 ≤ -3 → 매도 후보
- 그 사이: 관망

## 출력
- timestamp, symbol, score, components(JSON), action
- DB `signals` 테이블 저장

## 백테스트 (Phase 4)
- 위 가중치는 워크포워드 최적화 대상
- OOS 검증 필수
