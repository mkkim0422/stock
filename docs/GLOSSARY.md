# GLOSSARY — 용어집

| 용어 | 의미 |
|---|---|
| 스윙 트레이딩 | 며칠~몇 주 보유 매매 |
| 페이퍼 트레이딩 | 가상매매 (실주문 없음) |
| 슬리피지 | 호가-체결가 차이 |
| 호가 단위 | 가격 최소 변동 폭 (tick size) |
| 평단가 | 평균 매수단가 |
| 골든크로스 | 단기 MA가 장기 MA 상향 돌파 |
| 데드크로스 | 단기 MA가 장기 MA 하향 돌파 |
| RSI | 상대강도지수, 0~100 |
| MACD | 이동평균 수렴/발산 |
| KOSPI | 한국 종합주가지수 |
| KOSDAQ | 코스닥 종합지수 |
| ETF | 상장지수펀드 |
| 액면분할 | 1주를 N주로 쪼개기 (가격 1/N) |
| 거래세 | 매도 시 부과되는 세금 |
| OHLCV | 시고저종거 (Open/High/Low/Close/Volume) |
| Look-ahead bias | 미래 데이터 의도치 않은 사용 |
| Survivorship bias | 생존 종목만 보는 편향 |
| OOS | Out-of-Sample, 표본 외 검증 |
| WF | Walk-Forward, 슬라이딩 검증 |
| WAL | Write-Ahead Logging (SQLite) |
| KST | 한국 표준시 (UTC+9) |
| ET | 미국 동부시 (서머타임 변동) |
